Local-first AI terminal assistant

Ship code with an AI that lives in your terminal

lint-cli is a Node.js command-line tool that connects to a local OpenWebUI or Ollama-compatible API. It can read, write and list files in your current project, acting like a focused coding partner inside your terminal.

Core features

  • Interactive REPL-style prompt
  • File read, write and list tools
  • Project-scoped memory per directory
  • Local model over HTTP

Great for

  • Scaffolding small features
  • Editing existing JavaScript files
  • Generating tests and pages
  • Quick refactors in a repo

Requirements

  • Node.js 18 or newer
  • OpenWebUI or Ollama-compatible HTTP API
  • Default endpoint: http://localhost:11434/ollama/api/chat

Install and run 

npm install
npm start

# or globally
npm install -g .
lint-cli

Configure

In the project where you run the CLI, create a .env file: 

OLLAMA_API=http://localhost:11434
OLLAMA_API_KEY=your-api-key
